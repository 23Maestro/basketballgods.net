# Assets (optional Envato)

Drop licensed court art here to upgrade Scene 2.

## Expected paths

| Path | Use |
|------|-----|
| `assets/court/halfcourt.png` | Half-court top-down texture under Scene 2 scatter |

### `halfcourt.png` tips
- Top-down half court, basket at the **bottom** of the image
- Transparent or dark background preferred
- ~1000–2000 px wide is enough
- App auto-detects the file; no code change needed

Scene 1 (3D Plotly) uses drawn geometry only — photo textures do not map cleanly onto Plotly 3D.
